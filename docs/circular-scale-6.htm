Diagram Constructor

The constructor CircularScale creates an SVG diagram control object. Drawing elements are then built with methods. 

let diagram = CircularScale( "svg-001" );

Initializes the diagramOptions member with defaults from the moduleOptions and values supplied via diagramOptions parameter.

Parameters

target
String or Object. The ID string of the target SVG element or a reference to the target element. Mandatory. Default value is undefined.
diagramOptions
Object. Members supplied here override the module options. Optional. Default = { }.

Drawing Commands

addCircularPath

Parameters

type
String, mandatory, allowed values are "circle" and "spiral", default value is "circle". Determines whether the path is created as closed circle or open spiral. The spiral can make more than one turn, which can be useful to show chord extensions in the second octave.
